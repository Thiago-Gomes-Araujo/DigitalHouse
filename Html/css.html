Ame, como se ninguém nunca houvesse feito você sofrer. Trabalhe, como se não precisasse do dinheiro. Dance, como se ninguém estivesse olhando. Cante, como se ninguém estivesse ouvindo. Viva, como se fosse no paraíso!

Curta o que de melhor a vida lhe oferece com toda intensidade, como se fosse o último dia da sua vida. A vida muitas vezes é curta, mas mesmo assim seu caminho é longo.

Nela aprendemos a sorrir, chorar, amar, sofrer e a renascer para amanhecer e termos um lindo dia.
